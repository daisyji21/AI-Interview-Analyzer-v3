# 🤖 AI Interview Analyzer v3

## Multimodal AI-Based Interview Evaluation System

AI Interview Analyzer v3 is a deep learning and NLP-powered system that evaluates interview performance using speech analysis, natural language processing, emotion detection, and computer vision.

The project combines Transformer models, Speech AI, and Deep Learning techniques to generate intelligent interview feedback and performance analytics.

---

# 🚀 Features

## 🎤 Speech & Audio Analysis
- Speech-to-text transcription using Whisper
- Speech confidence analysis
- Speech pace detection
- Filler word detection

## 🧠 NLP & Transformer Models
- Sentiment analysis using BERT
- Semantic similarity using Sentence-BERT
- Vocabulary diversity analysis
- Answer relevance scoring

## 😊 Emotion & Visual Analysis
- Facial emotion detection using DeepFace
- Eye contact analysis using MediaPipe/OpenCV

## 📊 AI Performance Scoring
- Overall interview score generation
- Grade prediction system
- AI-generated feedback & recommendations

## 📄 Interactive Dashboard
- Modern AI report interface
- Interview transcript display
- Performance metrics visualization
- Dynamic recommendations

---

# 🏗️ System Architecture

Interview Video/Audio  
        ↓  
Speech Recognition (Whisper)  
        ↓  
NLP Analysis (BERT + Sentence-BERT)  
        ↓  
Emotion Detection (DeepFace)  
        ↓  
Eye Contact Analysis (MediaPipe)  
        ↓  
AI Scoring Engine  
        ↓  
Final Interview Evaluation Report

# ⚙️ Installation

Install required dependencies:

```bash
pip install openai-whisper
pip install transformers
pip install sentence-transformers
pip install deepface
pip install mediapipe
pip install librosa
pip install opencv-python
pip install pandas
pip install numpy
pip install plotly
```

# ▶️ How to Run

1. Open the notebook in Google Colab
2. Enable GPU:
   Runtime → Change Runtime Type → GPU
3. Run all cells sequentially
4. Upload interview video/audio
5. Generate AI interview report
